# Changelog

All notable changes to the Separator component will be documented in this file.

## [2026-01-03]

### Added

- feat(separator): add `children` prop support for displaying content between separator lines
- feat(separator): support "OR" divider pattern and section title use cases
- feat(separator): add `hasChildren` variant with compound variants for flexible layout

### Changed

- refactor(separator): restructure component to render two separator lines with content in between when children provided
- refactor(separator): update Tailwind Variants slots (`root`, `separator`) for better composition
- refactor(separator): use compoundVariants for orientation-specific layout when children are present
